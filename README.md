# theseus-continuity-framework

**Reproducibility package for**
**Intent-Aware Continuity for Evolving AI: Typed Adjudication with Intrinsic Fingerprinting for Verifiable Provenance**
(Matthew A. Davis, March 2026 — arXiv forthcoming)

[![Python](https://img.shields.io/badge/Python-3.10%2B-blue)](https://www.python.org)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

## Quick Start (60 seconds)
```bash
git clone https://github.com/MatthewDavisAIArchitect/theseus-continuity-framework.git
cd theseus-continuity-framework
cd fingerprint-anchor
pip install -r requirements.txt
bash run_harness.sh
```

## Repository Structure

* `fingerprint-anchor/` — HonestAGI-style intrinsic fingerprint + 10⁵-attack test harness
* `simulation/` — Stage-I Configuration Drift, metric *d*, repair operator *R*, stability balls
* `figures/` — All 6 figures from the manuscript (ready for LaTeX)
* `results/` — Output logs and plots generated by `run_all.py`
* `notebooks/` — Exploratory Jupyter demos (optional)

## What's New in This Version (matches revised arXiv paper)

* Concrete metric *d* implementation (weighted cosine + KL)
* Public foundation model framing (Grok-1, Llama-3, Claude)
* Full proofs of Theorem 1 & 3 reproduced in code comments
* Comparison to Kirchenbauer (2023) & Christ (2024) watermarking
* Explicit Limitations section (mirrors paper)

## One-Command Reproduction

```bash
bash run_harness.sh          # fingerprint anchor verification
python run_all.py            # runs everything + saves figures
```

## Limitations (see LIMITATIONS.md and paper Section 10)

* Synthetic logits only (no real 314B-scale fine-tuning yet)
* No adversarial subtraction of φ(x) tested
* Metric *d* weights are application-specific
* No MLOps integration (MLflow/DVC/W&B) yet
